# Shortcuts for Windows Terminal

## Directory

The `settings.json` is stored here

    $env:USERPROFILE\AppData\Local\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState

## Command Palette

    ctrl + shift + p

## New Tab

    ctrl + shift + 1/.../3

## Close Tab

    ctrl + w

## Search

    ctrl + f

## Jump to Tab 1-9

    ctrl + 1/.../9

## Jump to right Tab

    ctrl+ tab

## Jump to left Tab

    ctrl + shift + tab

## New Pane

    ctrl + shift + t

## New Pane vertical

    alt + shift + +

## New Pane horizontal

    alt + shift + -

## Close Pane

    ctrl + shift + w

## Change Pane Focus

    alt + leftarrow/rightarrow/uparrow/downarrow

## Change Pane Size

    alt + shift + leftarrow/rightarrow/uparrow/downarrow
